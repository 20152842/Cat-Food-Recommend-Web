# 🐱 냥이 사료 추천기

> **우리 고양이에게 딱 맞는 사료를 찾아주는 서비스예요!**

---

## 이 서비스가 뭔가요?

고양이 사료를 고를 때 "이 사료가 우리 고양이한테 맞나요?", "얼마나 줘야 하나요?", "예산 안에서 제일 좋은 게 뭔가요?" 같은 고민을 해본 적 있으신가요?

**냥이 사료 추천기**는 간단한 정보만 입력하면:

- 🔢 우리 고양이에게 필요한 **하루 칼로리**를 계산해 주고
- 💰 정해둔 **예산 안에서** 가장 좋은 사료를 골라주는 서비스예요

---

## 어떻게 사용하나요?

### 1단계: 고양이 정보 입력

아래 다섯 가지 정보를 입력해 주세요.

| 입력 항목 | 예시 |
|---|---|
| 몸무게 | 4.0 kg |
| 나이 | 3살 0개월 |
| 성별 | 수컷 |
| 중성화 여부 | 중성화 했음 |
| 월 사료 예산 | 50,000원 |

> 잘 모르겠다면 페이지 안에 있는 **예시 입력** 버튼을 눌러보세요!

### 2단계: 추천받기 버튼 클릭

**🔍 사료 추천받기** 버튼을 누르면 바로 결과가 나와요.

### 3단계: 결과 확인

- 우리 고양이의 **하루 권장 칼로리**를 알려줘요
- 예산 안에서 **단백질 함량이 높고 가성비 좋은 사료** 최대 5개를 추천해 줘요
- 각 사료마다 **하루에 몇 그램** 줘야 하는지, **하루/월 비용**이 얼마인지도 보여줘요

---

## 칼로리는 어떻게 계산하나요?

**미국 국립연구위원회(NRC)**와 **미국반려동물협회(AAFCO)**에서 발표한 공식을 사용해요.

```
하루 권장 칼로리 = 기초대사량 × 생활 단계 계수

기초대사량 (RER) = 70 × 체중(kg)^0.75
```

생활 단계에 따라 계수가 달라져요:

| 고양이 상태 | 계수 |
|---|---|
| 아기 고양이 (0~3개월) | 3.0 |
| 성장기 고양이 (4~11개월) | 2.5 |
| 성체 중성화 수컷 | 1.6 |
| 성체 중성화 암컷 | 1.4 |
| 성체 미중성화 수컷 | 1.8 |
| 성체 미중성화 암컷 | 1.6 |
| 노령묘 (7세 이상) | 1.4 |

---

## 추천 사료는 어떻게 고르나요?

두 가지 기준으로 점수를 매겨요:

1. **단백질 함량 (60%)**: 고양이는 육식 동물이라 단백질이 가장 중요해요
2. **예산 여유 (40%)**: 예산 대비 저렴할수록 점수가 올라가요

결과에는 **건식(일반 사료)**과 **습식(캔/파우치)** 모두 포함돼요.

---

## 주의사항

- 추천 사료는 참고용이에요. 실제로 바꾸기 전에 **수의사 선생님과 상담**하시는 걸 권장해요
- 고양이마다 기호성(좋아하는 맛)이 달라서 추천 1위 사료라도 안 먹을 수 있어요
- 사료 가격은 국내 유통 평균가 기준이며 실제 구매 가격과 다를 수 있어요

## 로컬 컴퓨터에서 직접 실행하는 방법

본인 PC에서 서비스를 돌려 보고 싶다면 아래 순서대로 진행하면 됩니다.

### 1. 필요한 프로그램 설치

| 프로그램 | 용도 | 받는 곳 |
|---------|------|--------|
| **Java 17 (JDK)** | 서버 실행에 필요 | [Adoptium Temurin 17](https://adoptium.net/temurin/releases/?version=17&os=Windows) 등에서 설치 |
| **Maven** | 빌드에 필요 | [Maven 다운로드](https://maven.apache.org/download.cgi) → 압축 풀고, 설치 가이드대로 경로 설정 |

> Java만 설치한 뒤, **명령 프롬프트**나 **PowerShell**에서 `java -version`, `mvn -v` 를 입력해 각각 버전이 나오면 준비 완료예요.

### 2. 프로젝트 받기

- **Git 사용하는 경우**: `git clone https://github.com/20152842/Cat-Food-Recommend-Web.git` 후 해당 폴더로 이동  
- **ZIP으로 받는 경우**: GitHub에서 **Code → Download ZIP** 으로 받고 압축을 푼 폴더로 이동

### 3. 터미널에서 실행

프로젝트 **폴더 안**에서 아래 명령을 실행하세요.

**Windows (명령 프롬프트 / PowerShell):**
```text
mvn spring-boot:run
```

**Mac / Linux (Maven 설치 후):**
```text
mvn spring-boot:run
```

처음 실행 시 라이브러리를 받느라 1~2분 걸릴 수 있어요.  
`Started ... CatFoodRecommendApplication` 같은 문구가 보이면 **실행된 것**이에요.

### 4. 브라우저에서 접속

실행이 되었다면 웹 브라우저 주소창에 아래를 입력하세요.

```text
http://localhost:8080
```

이제 로컬에서 냥이 사료 추천기를 사용할 수 있어요.

### 참고

- **네이버 쇼핑 검색**: 실제 상품 검색 결과를 쓰려면 개발자용 설정(`application.properties`의 네이버 API 키)이 필요해요. 없어도 **칼로리 계산**과 **비교 목록** 기능은 그대로 사용할 수 있어요.
- **종료**: 터미널 창에서 `Ctrl + C` 를 누르면 서버가 종료돼요.
